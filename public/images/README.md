# 图片资源说明

## 需要准备的图片

### 首页 (Hero)
- **kailash-hero.jpg** - 冈仁波齐全景大图（16:9，建议 1920x1080 以上）
- 用途：首页顶部背景
- 备选：可用渐变色背景替代

### 神山文化页
- **kailash-culture.jpg** - 神山文化相关图片
- **religion-*.jpg** - 四大宗教相关图片（可选）

### 转山路线页
- **route-map.jpg** - 转山路线图
- 建议：手绘或标注版地图

### 预订页面
- **package-*.jpg** - 套餐展示图片（可选）

### 通用
- **og-image.jpg** - Open Graph 分享图（1200x630）
- **favicon.svg** - 网站图标
- **apple-touch-icon.png** - iOS 图标（180x180）

## 图片来源建议

1. **自己拍摄** - 如果去过冈仁波齐，用自己的照片最独特
2. **免费图库** - Unsplash、Pixabay 搜索 "Kailash"、"Tibet"、"Mountain"
3. **购买授权** - 500px、Getty Images

## 图片优化

所有图片上传到网站前应该：
1. 压缩（TinyPNG、Squoosh）
2. 转换为 WebP 格式（可选但推荐）
3. 提供多种尺寸用于响应式

## 占位符

在添加真实图片前，网站使用：
- 渐变色背景
- 图标占位
- 文字描述
