## 图片来源建议

### 免费图库
1. **Unsplash** (unsplash.com) - 搜索 "Mount Kailash", "Tibet", "Pilgrimage"
2. **Pexels** (pexels.com)
3. **Pixabay** (pixabay.com)

### 注意事项
- 选择高清图片（建议1920x1080以上）
- 注意版权，优先使用CC0协议图片
- 图片大小控制在500KB以内（网页加载速度）
- 命名规范：`页面名-描述.jpg`，如 `hero-kailash-sunrise.jpg`

### 图片处理
- 格式：WebP 优先，JPEG 备选
- 压缩工具：TinyPNG、Squoosh
- 尺寸：提供多种尺寸适配不同设备
